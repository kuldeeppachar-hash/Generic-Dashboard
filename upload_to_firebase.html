⬆ Generic Division — Firebase Data Uploader

Upload the raw SAP export → it filters to Division = "Generic", aggregates, and pushes live to Firebase. No SO data is generated or stored.
// SOURCE OF TRUTH
This tool reads only the raw SAP-derived export Excel — nothing else feeds this dashboard.
It auto-detects the data sheet (some exports have blank scratch sheets before the real one) and the column format, then filters to the Generic division only.
Two export formats are supported: SAP direct export (Division Name = "Generic"), and distributor-type export (Distribtor Type = "Generic Distributors" and Product type = "Other-Generic"). "City" is read from City or Head Quarter, whichever the file has.
// MONTH-BY-MONTH STORAGE
Each upload is detected by its Invoice Date and saved under its own month — it no longer overwrites other months.
Re-uploading a file for a month you've already uploaded will overwrite that month only (useful for corrections); everything else stays untouched.
The dashboard reads all months and lets you compare them side by side.
